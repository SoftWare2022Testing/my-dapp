WalletConnect dApp (built by WalletConnect v2 + Ethers.js)

Since Vercel does not support the deployments of dynamic web pages, user should download our DApp (for WalletConnect protocol) source code and run it locally.

1. Download our Dapp code (for WalletConnect):

Click here for download

2. Install the dapp's dependencies:

yarn

3. Modify the environment variables:

vim .env.local (in dapp's root directoy)

4. Build the project

NEXT_PUBLIC_PROJECT_ID - generate your own ProjectId at https://cloud.walletconnect.com NEXT_PUBLIC_RELAY_URL (already set)

5. Build the project

npm run build npm run start

6. visit url: http://localhost:3000
